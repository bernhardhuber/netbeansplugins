About jdepend

This module calculates the package dependencies of a project.

For calculating the package dependencies, the project JDepend is used; see:

Usage

For viewing the package dependencies do the following steps:

  • Select your project in the project explorer window
  • Select Build|JDepend
  • Your selected browser opens, and displays the package dependencies

Options

You can set following options

  • Enable analyzing inner classes.
  • Define package filters. Packages matching these filters are not analyzed. You can add and remove package filters. Duplicate package filters are not allowed.
  • Reset to the default package filters java.*, and javax.*.
  • Select an xslt file for rendering the jdepend result document.

You can set these options via Tools|Options|Miscellaneous|JDepend.
