About nb-cpd

This module integrates PMD's Copy/Paste Detector (CPD) tool.

CPD can find duplicate code in your project. Read more about CPD at the PMD's homepage:


Usage

For code duplicates do the following steps:

  • Select your files, or your source directory in the project, files, or favorites explorer window
  • Select Build|PMD's Copy/Paste Detector
  • In the output tab CPD Result the result of running CPD is shown.

For text, and xml output formats the lines specifying files having code duplication are hyperlinked. Clicking the line opens the file.

In output window you can navigate to the next file by Navigate|Next Error (F12), and to the previous file by Navigate|Previous Error (Shift+F12).

Options

You can set following options

  • Scan Directory Recursively - Enable scanning of subdirectories.
  • Minimal Token Count - Minimal count of tokens that must match for detecting code duplication.
  • Language - Process only files of specific code language
  • Extension - For the language selection 'by_extension" specify the file extension
  • Ignore Identifiers, Ignore Literals - For the java language CPD allows you to ignore identifiers, and literals when doing the processing.
  • Renderer Output - select the renderer output format.

You can set these options via Tools|Options|Miscellaneous|PMD's Copy/Paste Detector.
