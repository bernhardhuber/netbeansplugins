About httppost

This module helps you posting ad-hoc http-requests.

Usage

The most simple way to use the http-post module do the following steps:

  1. Open the window HttpPost, using Windows|HttpPost.
  2. Enter the URL details into form of the http-post window.
  3. Press button HttpPost in the toolbar of the http-post window.

Pressing HttpPost sends the URL in the http-post window. The response section of the http-post window displays the response information.
Moreover the tree view shows the sent URL in the folder Current.

Advanced Usage

The tree view of the http-post window displays two folder nodes:

  • Current
  • Saved

The folder Current displays all URLs posted during a user session. Nodes in this folder offers following node actions:

Edit
Load the node information into the form of the http-post window.
Delete
Remove the node from this folder.
Save
Move the node to the folder Saved.
Properties
Displays a dialog of the URL properties

The folder Saved displays all URLs saved to the filesystem. During startup saved URLs are loaded into this folder. Nodes in this folder offers following node actions:

Edit
Load the node information into the form of the http-post window
Delete
Remove the node from this folder, and the filesystem
Properties
Displays a dialog of the URL properties

Moreover the properties of the currently selected node are shown in properties window.

Details

Persisted URLs are stored in the filesystem at location {nbuser.dir}/config/httppost.
The name of the files are calculated from the start-date, and end-date of the URL post.
The data format of the files are the format as defined by java.beans.XMLEncoder, and java.beans.XMLDecoder.
