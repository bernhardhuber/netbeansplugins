Rule CouplingBetweenObjects::

TOC

Description

This rule counts unique attributes, local variables and return types within an object. A number
 higher than specified threshold can indicate a high degree of coupling.
Message
High amount of different objects as members denotes a high coupling
Class
net.sourceforge.pmd.rules.CouplingBetweenObjects
Reference
-
ExternalInfoUrl
http://pmd.sourceforge.net/rules/coupling.html#CouplingBetweenObjects
TypeResolution
-
Priority
3

Example

Example code snippet of this rule. 

import com.Blah;
import org.Bar;
import org.Bardo;
public class Foo {
 private Blah var1;
 private Bar var2;
 //followed by many imports of unique objects
 void ObjectC doWork() {
  Bardo var55;
  ObjectA var44;
  ObjectZ var93;
  return something;
 }
}

Properties

Following properties are defined:

threshold
Description
The unique type reporting threshold

Rule ExcessiveImports::

TOC

Description

A high number of imports can indicate a high degree of coupling within
an object. Rule counts the number of unique imports and reports a violation
if the count is above the user defined threshold.
Message
A high number of imports can indicate a high degree of coupling within an object.
Class
net.sourceforge.pmd.rules.ExcessiveImports
Reference
-
ExternalInfoUrl
http://pmd.sourceforge.net/rules/coupling.html#ExcessiveImports
TypeResolution
-
Priority
3

Example

Example code snippet of this rule. 
      
import blah.blah.Baz;
import blah.blah.Bif;
// 18 others from the same package elided
public class Foo {
 public void doWork() {}
}

Description
The import count reporting threshold

Rule LooseCoupling::

TOC

Description

Avoid using implementation types (i.e., HashSet); use the interface (i.e, Set) instead
Message
Avoid using implementation types like ''{0}''; use the interface instead
Class
net.sourceforge.pmd.rules.design.LooseCoupling
Reference
-
ExternalInfoUrl
http://pmd.sourceforge.net/rules/coupling.html#LooseCoupling
TypeResolution
-
Priority
3

Example

Example code snippet of this rule. 

import java.util.*;
public class Bar {
 // Use List instead
 private ArrayList list = new ArrayList();
 // Use Set instead
 public HashSet getFoo() {
  return new HashSet();
 }
}
