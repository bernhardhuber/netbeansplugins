Rule DontImportJavaLang::

TOC

Description

Avoid importing anything from the package 'java.lang'.  These classes are automatically imported (JLS 7.5.3).
Message
Avoid importing anything from the package 'java.lang'
Class
net.sourceforge.pmd.rules.imports.DontImportJavaLang
Reference
-
ExternalInfoUrl
http://pmd.sourceforge.net/rules/imports.html#DontImportJavaLang
TypeResolution
-
Priority
4

Example

Example code snippet of this rule. 

// this is bad
import java.lang.String;
public class Foo {}

// --- in another source code file...

// this is bad
import java.lang.*;

public class Foo {}
