Uncompress Gzip

The Gzip Module allows to uncompress selected files using the GZIP format.

Uncompressing Using Gzip

Do the following steps for uncompressing a file:

  • Select one or more files having the extension .gzin an explorer view
  • Invoke Tools|Gzip either from the popup menu, or from the main menu
  • Answer the dialog if you want to uncompress the file
  • Answer the dialog if you want to delete the original file
