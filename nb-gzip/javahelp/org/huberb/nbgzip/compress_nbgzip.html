Compress Gzip

The Gzip Module allows to compress selected files using the GZIP format.

Compressing Using Gzip

Do the following steps for compressing a file:

  • Select one or more files in an explorer view
  • Invoke Tools|Gzip either from the popup menu, or from the main menu
  • Answer the dialog if you want to compress the file
  • Answer the dialog if you want to delete the original file
